- 点击后打开接口搜索面板，快捷搜索需要调用的接口，选择接口后生成调用的代码片段。
- 通过重写 `CodeGenerator.codeSnippet `，自定义需要生成代码片段。具体使用，请参考 pont-config.json 配置项
